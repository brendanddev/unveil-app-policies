# Server-To-Client-Scripts

# Overview
This repository contains three PHP scripts tackling unique challenges involving ants and number theory.

### 1. Marching Ants Game
This script simulates a battle between two ant colonies represented by a string containing 'R' (red ants), 'B' (black ants), and 'X' (empty spaces). The script determines the outcome based on encounters between ants and outputs whether neither colony survives, one wins, or both are eliminated.

### 2. Basic Factoring
This script finds and returns all factors of a given natural number 𝑁. It handles errors for negative or invalid inputs.

### 3. Erdős–Woods Numbers
This script identifies numbers within a specified range that are not factored by the prime factors of two input numbers. It checks for valid input and returns results accordingly.

#### Getting Started
To use these scripts, clone the repository and run them on a PHP server like WAMP, XAMPP, or LAMP. Adjust your requests as needed to interact with the functionality provided.
